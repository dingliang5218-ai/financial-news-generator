## 🚀 快速开始

### 方式 1: 快速启动脚本

```bash
./run.sh
```

### 方式 2: 手动启动

```bash
# 1. 配置环境
cp .env.example .env
# 编辑 .env，添加 CLAUDE_API_KEY

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行测试
python test_system.py

# 4. 启动系统
python main.py
```

### 方式 3: Docker 部署

```bash
docker-compose up -d
```
