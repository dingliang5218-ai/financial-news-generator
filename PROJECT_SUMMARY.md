## 使用方法

### 快速启动

```bash
# 1. 配置环境变量
cp .env.example .env
# 编辑 .env 添加 CLAUDE_API_KEY

# 2. 运行快速启动脚本
./run.sh

# 3. 启动系统
python main.py
```

### 测试运行

```bash
# 运行集成测试
python test_system.py

# 运行健康检查
python main.py --health-check

# 单次生成测试
python main.py --run-once daily
```

## 注意事项

1. **API Key**: 需要有效的 Claude API Key
2. **RSS 源**: 需要在 .env 中配置 RSS_FEEDS
3. **磁盘空间**: 至少需要 100MB 可用空间
4. **Python 版本**: 需要 Python 3.8 或更高版本
